# Nmapе®һж—¶дё»жңәжҺўжөӢ

  1. д»Җд№ҲжҳҜnmap
    NmapпјҲNetwork MapperпјүжҳҜдёҖдёӘејҖжәҗзҡ„зҪ‘з»ңжү«жҸҸе·Ҙе…·пјҢз”ЁдәҺжҺўжөӢзҪ‘з»ңдёҠзҡ„дё»жңәгҖҒжңҚеҠЎе’ҢзҪ‘з»ңеҸҜз”ЁжҖ§гҖӮе®ғе…·жңүе№ҝжіӣзҡ„еҠҹиғҪе’ҢзҒөжҙ»жҖ§пјҢеҸҜз”ЁдәҺеӨҡз§ҚзҪ‘з»ңзӣёе…ід»»еҠЎпјҢеҢ…жӢ¬пјҡ
  • дё»жңәеҸ‘зҺ°пјҡиҜҶеҲ«зҪ‘з»ңдёҠжҙ»еҠЁзҡ„дё»жңәгҖӮ
  • з«ҜеҸЈжү«жҸҸпјҡзЎ®е®ҡдё»жңәдёҠејҖж”ҫзҡ„з«ҜеҸЈе’ҢжӯЈеңЁиҝҗиЎҢзҡ„жңҚеҠЎгҖӮ
  • жңҚеҠЎзүҲжң¬жЈҖжөӢпјҡе°қиҜ•иҜҶеҲ«дё»жңәдёҠиҝҗиЎҢзҡ„е…·дҪ“жңҚеҠЎе’Ңеә”з”ЁзЁӢеәҸзҡ„зүҲжң¬дҝЎжҒҜгҖӮ
  • ж“ҚдҪңзі»з»ҹжЈҖжөӢпјҡе°қиҜ•зЎ®е®ҡзӣ®ж Үдё»жңәзҡ„ж“ҚдҪңзі»з»ҹзұ»еһӢе’ҢзүҲжң¬гҖӮ
  • и„ҡжң¬жү«жҸҸпјҡдҪҝз”ЁNmapи„ҡжң¬еј•ж“Һжү§иЎҢиҮӘе®ҡд№үи„ҡжң¬пјҢеҜ№дё»жңәиҝӣиЎҢжӣҙж·ұе…Ҙзҡ„жЈҖжөӢе’ҢиҜ„дј°гҖӮ

Nmapеёёиў«зҪ‘з»ңз®ЎзҗҶе‘ҳгҖҒе®үе…Ёдё“е®¶е’Ңжё—йҖҸжөӢиҜ•дәәе‘ҳз”ЁдәҺзҪ‘з»ңиҜ„дј°гҖҒе®үе…Ёе®Ўи®Ўе’Ңзӣ‘жҺ§гҖӮдҪҶжҳҜпјҢз”ұдәҺе…¶еҠҹиғҪејәеӨ§пјҢдҪҝз”ЁиҖ…йңҖиҰҒи°Ёж…ҺдҪҝз”Ёе№¶йҒөе®Ҳжі•еҫӢжі•и§„пјҢзЎ®дҝқеңЁеҗҲжі•е’ҢжҺҲжқғзҡ„иҢғеӣҙеҶ…дҪҝз”ЁNmapиҝӣиЎҢзҪ‘з»ңжү«жҸҸе’ҢжөӢиҜ•гҖӮ

  1. Nmap е‘Ҫд»ӨиЎҢйҖүйЎ№зҡ„еҝ«йҖҹж‘ҳиҰҒ
  • жү«жҸҸзұ»еһӢдёҺе‘Ҫд»ӨзӨәдҫӢ
жү«жҸҸзұ»еһӢе‘Ҫд»ӨзӨәдҫӢиҜҙжҳҺ
ARPжү«жҸҸsudo nmap -PR -sn MACHINE_IP/24иҝҷдёӘйҖүйЎ№е‘ҠиҜү nmap дҪҝз”ЁICMPеҚҸи®®зҡ„PingпјҲARPиҜ·жұӮеҢ…пјүиҜ·жұӮпјҢеҜ№зӣ®ж ҮзҪ‘з»ңзҡ„дё»жңәиҝӣиЎҢдё»жңәеҸ‘зҺ°
ICMP еӣһжіўжү«жҸҸsudo nmap -PE -sn MACHINE_IP/24е‘ҠиҜү nmap дҪҝз”ЁICMPеҚҸи®®зҡ„EchoиҜ·жұӮпјҲpingпјүпјҢеҜ№зӣ®ж ҮзҪ‘з»ңзҡ„дё»жңәиҝӣиЎҢдё»жңәеҸ‘зҺ°
ICMP ж—¶й—ҙжҲіжү«жҸҸsudo nmap -PP -sn MACHINE_IP/24
ICMP ең°еқҖжҺ©з Ғжү«жҸҸsudo nmap -PM -sn MACHINE_IP/24
TCP SYN Ping жү«жҸҸsudo nmap -PS22,80,443 -sn MACHINE_IP/30
TCP ACK Ping жү«жҸҸsudo nmap -PA22,80,443 -sn MACHINE_IP/30
UDP Ping жү«жҸҸsudo nmap -PU53,161,162 -sn MACHINE_IP/30
SYN жү«жҸҸпјҲеҚійҡҗз§ҳжү«жҸҸпјүnmap -sS 10.10.231.61иҝҷдёӘе‘Ҫд»Өдёӯзҡ„еҸӮж•° "-sS" жҢҮе®ҡдҪҝз”Ё SYN жү«жҸҸгҖӮSYN жү«жҸҸжҳҜдёҖз§Қ TCP жү«жҸҸжҠҖжңҜпјҢе®ғйҖҡиҝҮеҸ‘йҖҒ TCP зҡ„ SYN еҢ…жқҘе°қиҜ•иҝһжҺҘеҲ°зӣ®ж Үдё»жңәзҡ„з«ҜеҸЈпјҢз„¶еҗҺж №жҚ®е“Қеә”жқҘеҲӨж–ӯз«ҜеҸЈжҳҜеҗҰејҖж”ҫгҖӮ
TCPиҝһжҺҘжү«жҸҸпјҲ-sTпјүnmap -sT 10.10.231.61жҳҜдёҖз§ҚеҹәдәҺTCPеҚҸи®®зҡ„жү«жҸҸж–№ејҸпјҢе®ғйҖҡиҝҮеҗ‘зӣ®ж Үдё»жңәзҡ„жҜҸдёӘз«ҜеҸЈеҸ‘йҖҒе®Ңж•ҙзҡ„TCPиҝһжҺҘиҜ·жұӮжқҘжү«жҸҸз«ҜеҸЈгҖӮиҝҷз§Қжү«жҸҸж–№ејҸзҡ„иЎҢдёәзұ»дјјдәҺжӯЈеёёзҡ„TCPиҝһжҺҘпјҢеӣ жӯӨеҸҜиғҪиў«зӣ®ж Үдё»жңәзҡ„йҳІзҒ«еўҷжҲ–е…ҘдҫөжЈҖжөӢзі»з»ҹжүҖи®°еҪ•гҖӮ
UDPжү«жҸҸ пјҲ-sUпјү-sU: иҝҷдёӘйҖүйЎ№е‘ҠиҜүNmapжү§иЎҢUDPжү«жҸҸгҖӮUDPжҳҜдёҖз§Қж— иҝһжҺҘзҡ„дј иҫ“еҚҸи®®пјҢдёҺTCPпјҲжңүиҝһжҺҘпјүдёҚеҗҢгҖӮUDPжү«жҸҸе°қиҜ•еҸ‘зҺ°зӣ®ж Үдё»жңәдёҠејҖж”ҫзҡ„UDPз«ҜеҸЈгҖӮ-F: иҝҷдёӘйҖүйЎ№е‘ҠиҜүNmapжү§иЎҢеҝ«йҖҹжү«жҸҸпјҲеҝ«йҖҹжЁЎејҸпјүпјҢеҸӘжү«жҸҸжңҖеёёи§Ғзҡ„з«ҜеҸЈгҖӮеҜ№дәҺUDPжү«жҸҸпјҢе®ғдјҡе°қиҜ•жү«жҸҸдёҖз»„еёёи§Ғзҡ„UDPз«ҜеҸЈиҖҢдёҚжҳҜж•ҙдёӘз«ҜеҸЈиҢғеӣҙ,-v: иҝҷдёӘйҖүйЎ№еҗҜз”ЁдәҶиҜҰз»ҶжЁЎејҸпјҢиҫ“еҮәжӣҙеӨҡзҡ„дҝЎжҒҜпјҢеҢ…жӢ¬жү«жҸҸзҡ„иҝӣеәҰе’ҢжӣҙиҜҰз»Ҷзҡ„з»“жһң

-sn еҰӮжһңжӮЁеҸӘеҜ№дё»жңәеҸ‘зҺ°ж„ҹе…ҙи¶ЈиҖҢж— йңҖз«ҜеҸЈжү«жҸҸпјҢиҜ·и®°дҪҸж·»еҠ гҖӮзңҒз•Ҙ -sn е°ҶдҪҝ Nmap й»ҳи®ӨеҜ№жҙ»еҠЁдё»жңәиҝӣиЎҢз«ҜеҸЈжү«жҸҸгҖӮ

EchoиҜ·жұӮпјҲICMP Echoпјүе’ҢARPиҜ·жұӮеҢ…пјҲAddress Resolution ProtocolпјүжҳҜз”ЁдәҺзҪ‘з»ңйҖҡдҝЎзҡ„дёҚеҗҢзұ»еһӢзҡ„ж•°жҚ®еҢ…пјҢе…·жңүдёҚеҗҢзҡ„еҠҹиғҪе’ҢдҪңз”Ёпјҡ 1.ICMP EchoиҜ·жұӮпјҡ ICMPпјҲInternet Control Message ProtocolпјүжҳҜз”ЁдәҺеңЁIPзҪ‘з»ңдёҠеҸ‘йҖҒй”ҷиҜҜж¶ҲжҒҜе’Ңж“ҚдҪңж¶ҲжҒҜзҡ„еҚҸи®®гҖӮе…¶дёӯпјҢEchoиҜ·жұӮжҳҜICMPж¶ҲжҒҜзұ»еһӢд№ӢдёҖгҖӮ ICMP EchoиҜ·жұӮз”ЁдәҺжҺўжөӢиҝңзЁӢдё»жңәзҡ„еҸҜиҫҫжҖ§пјҢеҸ‘йҖҒEchoиҜ·жұӮе№¶зӯүеҫ…иҝңзЁӢдё»жңәеӣһеӨҚпјҢд»ҺиҖҢзЎ®и®Өдё»жңәжҳҜеҗҰеҸҜи®ҝй—®гҖӮ еңЁзҪ‘з»ңиҜҠж–ӯе’Ңдё»жңәеҸ‘зҺ°дёӯеёёз”ЁпјҢдҫӢеҰӮPingе‘Ҫд»ӨйҖҡеёёдҪҝз”ЁICMP EchoиҜ·жұӮгҖӮ 2.ARPиҜ·жұӮеҢ…пјҡ ARPпјҲAddress Resolution ProtocolпјүжҳҜз”ЁдәҺеңЁIPең°еқҖе’ҢMACең°еқҖд№Ӣй—ҙе»әз«Ӣжҳ е°„е…ізі»зҡ„еҚҸи®®гҖӮ ARPиҜ·жұӮеҢ…з”ЁдәҺеңЁеұҖеҹҹзҪ‘дёҠеҜ»жүҫзӣ®ж Үи®ҫеӨҮзҡ„MACең°еқҖгҖӮдё»жңәеңЁе°қиҜ•дёҺеҸҰдёҖеҸ°дё»жңәиҝӣиЎҢйҖҡдҝЎж—¶пјҢдјҡйҰ–е…ҲжЈҖжҹҘиҮӘе·ұзҡ„ARPзј“еӯҳпјҢеҰӮжһңжүҫдёҚеҲ°еҜ№еә”зҡ„MACең°еқҖпјҢеҲҷдјҡеҸ‘йҖҒARPиҜ·жұӮгҖӮ ARPиҜ·жұӮе№ҝж’ӯеҲ°зҪ‘з»ңдёҠзҡ„жүҖжңүи®ҫеӨҮпјҢиҜўй—®зү№е®ҡIPең°еқҖеҜ№еә”зҡ„MACең°еқҖпјҢиҜҘIPең°еқҖеҜ№еә”зҡ„и®ҫеӨҮдјҡеӣһеӨҚеҢ…еҗ«иҮӘе·ұMACең°еқҖзҡ„ARPе“Қеә”еҢ…гҖӮ жҖ»зҡ„жқҘиҜҙпјҢICMP EchoиҜ·жұӮз”ЁдәҺжЈҖжөӢдё»жңәзҡ„еҸҜиҫҫжҖ§пјҢиҖҢARPиҜ·жұӮеҢ…з”ЁдәҺеңЁеұҖеҹҹзҪ‘дёӯеҜ»жүҫIPең°еқҖеҜ№еә”зҡ„MACең°еқҖгҖӮе®ғд»¬йғҪжҳҜзҪ‘з»ңйҖҡдҝЎдёӯеёёз”Ёзҡ„ж•°жҚ®еҢ…зұ»еһӢпјҢдҪҶз”ЁйҖ”е’ҢеҠҹиғҪдёҚеҗҢгҖӮ
